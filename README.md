# Как запустить

npm install -> npm run dev -> http://localhost:5173

## Как обрабатывается стрим

- Файл - useStreamPlayer

- Создаем имитацию SSE с циклом for и задержкой sleep

- Проигрывание идёт с задержкой (delay) 50–150 мс (имитация реального стрима)

- token.delta постепенно склеивается в итоговый текст

- Делим delay значение на speed, для регулирования скорости воспроизведения, например delay = 100, speed = 1 -> 100 / 1 = 100

- StreamingOutput - обрабатываем множество json

## Как обрабатывается vega spec

- Файл - useVegaSpec

- Обработка пустых значений

- Достаем json из обратных скобок

- Минимальная валидация на mark & encoding

- Рендер с react-vega
